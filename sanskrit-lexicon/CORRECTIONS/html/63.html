Title - WIL corrections from alphabetical misorderings part 1

funderburkjim :

This issue looks for headword spelling errors drawn from a list of alphabetical misorderings in the Wilson dictionary.

This part 1 deals with cases 1-100 from a list of 417 cases.

Here is how the 417 cases were selected.

Start with the list of Wilson headwords (wilhw2.txt) available as part of the
xml downloads in the [Wilson downloads](http://www.sanskrit-lexicon.uni-koeln.de/scans/WILScan/2014/web/webtc/download.html)

Program [hw2_chksort1.py](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/hw2_chksort1.py) constructs a list of all alphabetical misorderings
in file [chksort1.txt](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/chksort1.txt) .A record in this file comes from the situation where the headword in a line of wilhw2 does not alphabetically precede the headword in the next line of wilhw2. chksort1.txt has 1189 lines of misorderings.

From prior work it is known that there are some genuine misorderings in Wilson dictionary which are not due to spelling errors. SInce the 1189 cases of chksort1 was deemed to be too large to handle all at once some method for filtering cases was sought.

The idea chosen was to look for cases where one of the two headwords involved in a misordering occurs ONLY in the Wilson dictionary and not as a headword in any of the other dictionaries of the
Cologne Sanskrit Lexicon. This piece of information is implicitly available in the
[sanhw1.txt] (https://github.com/sanskrit-lexicon/CORRECTIONS/blob/master/sanhw1/sanhw1.txt) file which represents headwords in all the dictionaries.

The program [chksort1a.py](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/chksort1a.py) carries out this filtering.
The program partitions chksort1.txt into two parts:
413 [more likely spelling errors](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/chksort1a.txt)
and 776 [less likely spelling errors](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/chksort1a_rest.txt) .
The list of 417 headword correction [candidates](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/chksort1a_singletons.txt) consists of the headwords involved in the list of more likely spelling errors.

Finally an [html](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/alpha1_ff.html)
page with links to Wilson for the 417 candidates was made to facilitate examination of the cases.

[Here](https://funderburkjim.github.io/wil_alpha1_ff.html) is the web page of links for anyone who may want to help.
