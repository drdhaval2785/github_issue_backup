funderburkjim :

I've changed one thing in the display of MW so now key2 for sArisTAKA displays with the candra-bindu.
Here's the comment and change made to the disp.php program (in webtc directory of the display for MW):
```
/* Mar 23 2015. Normally in MW ~ is used as a 'space' character
this is a holdover from a coding convention in Malten's Monier.all.
However in the current SLP1 ~ is used to indicate candra-bindu.
This appears in key2 for one word (key1=sArisTAKA key2=sArisTA-KA~);
As far as is known this is the only use of ~ in Sanskrit-SLP1 in
mw.xml. Thus we change this global replacement in all of $line
to deal just with the $body.
$line = preg_replace('/~/'' '$line);
*/
if (preg_match('|^(.*)(.*)(.*)$|'$line$matches)){
$head = $matches[1];
$body = $matches[2];
$tail = $matches[3];
$body = preg_replace('/~/'' '$body);
$line = $head . $body . $tail;
}
```

funderburkjim :

For our information here is my email question to Peter regarding M~ and his response:
```
Peter -
In PWG there are 577 records of pwg.xml in which the current coding of Devanagari in SLP1 includes
a ~. In almost all of these the form is VOWEL+M+~. Is the form M~ redundant in SLP1 i.e. should
all of these cases be changed to VOWEL+~ ?
Incidentally why is candra-bindu used at all - does it represent a different pronunciation than M?
Why is it absent from the usual listing of the Devanagari alphabet?
```

```
Dear Jim
\M~\ is not meaningful in SLP1. 'M' indicates anusvAra; '~' indicates nasalization which does not apply to
anusvAra since anusvAra is already a nasal sound.
I believe the endoding \M~\ is a relic of Thomas' original encoding which distinguished bindu from candra
bindu. In SKD at least bindu was encoded as M and candrabindu as MM. We should check whether \M~\
represents candrabindu as MM did in SKD.
Now the use of bindu and candrabindu in Devanagari Vedic texts varies so it is not clear what the phonetic
significance is without some further investigation. When it occurs above a vowel or semivowel y l or v
candra bindu represents nasalization of the vowel or semivowel. If candrabindu occurs with a virama (or
avagraha) below it it indicates an anusvAra. Many texts use bindu for an anusvAra that is susceptible to
being replaced by a stop homorganic with the following stop e.g. vanaM gacCati could become by sandhi
vanaN gacCati but contrast this with a 'true' anusvAra which only occurs before spirants (S z s h) and r
e.g. vanaM sarati vanaM rocate. If the use of M~ is consistent according to the above contexts it may be
possible to accurately transcode the 'M~' to either 'M' or '~' depending upon that context.
```
