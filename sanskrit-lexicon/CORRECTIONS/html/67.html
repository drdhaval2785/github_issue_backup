Title - WIL corrections from alphabetical misorderings part 5

funderburkjim :

This issue looks for headword spelling errors drawn from a list of 289 alphabetical misorderings in the Wilson dictionary. In contrast to the prior list of 417 misorderings most of these probably will not need correction. These are cases where both the misordered headwords appear in other dictionaries besides Wilson; there are 776 of these [less likely spelling errors](https://github.com/sanskrit-lexicon/WIL/blob/master/alphawork/chksort1a_rest.txt) .

The present list of 289 is drawn from the larger list as follows.

Each of the 776 misorderings involves a pair of consecutive headwords from the dictionary ordered list of Wilson headwords and it is true that the first of this pair of headwords does not alphabetically precede the second. Consider either one of these and call it X.

Next consider all the headwords Y appearing in ANY dictionary (i.e. Y is in the sanhw1.txt list)
and choose only those Y such that three conditions hold:
Y and X start with the same letter
Y and X are nearly the same (specifically the Levenshtein (or edit) distance between X and Y is 1 or 2.
If Y is used in place of X then the alphabetical ordering will be restored at this position in the headword list.

The set of those X for which there is one or more such Y is the list of 289 headwords being considered here. Note that furthermore Y provides us with one or more suggestions as to the
correction to make. Of course sometimes the suggestion Y will be wrong. And since X occurs
in more dictionaries than just Wilson Y will usually be wrong.

Nonethesless we will get some errors and reducing from 776 to 289 is a 2/3 reduction in cases to
examine.

In the following I will show details only for those cases where a spelling correction is found to be required.
