Title - Missing data in INM

funderburkjim :

There are 89 cases of missing data in the digitization of INM (Index to Names in the Mahabharata by S. Sorensen). These are detailed [here](https://github.com/sanskrit-lexicon/CORRECTIONS/blob/master/dictionaries/INM/missing_0001-0099.txt)

If anyone wants to tackle these I can supply the fairly simple instructions. Basically you have to look at
each case and by some means (such as examination of scan or the print version of text) determine what
the missing text should be and input that and your initials in the line **below** the <orig> line in the file above. When all the corrections are accounted for I have a program to process the adjusted file and install the corrections.

A first glance at a couple of examples suggests that this might be hard from our scans alone; so probably reference to a better-printed paper version will be needed -- but this is just a guess.

gasyoun :

I hardly believe that other printed copies will help. I have none but still in some cases I understand that what we need to do is decipherment using some kind of prior knowledge or logic. Slowly I can look at them to see if I can help at all. Not all cases are hopeless. As you will see in case of `agastya` - it's just some guesswork but the result is 100% correct.
